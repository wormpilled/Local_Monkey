2. Development Workflow

  1. Edit scripts in your preferred editor (VS Code, etc.)
  2. Save changes to /var/www/html/violent_monkey/aistudio/
  3. Refresh the target website - scripts reload automatically
  4. Check browser console for debugging info

3. Adding New Scripts

  1. Create your script file in the appropriate directory
  2. Add it to the CONFIG.scripts array in master.user.js
  3. Set conditions and triggers for when it should load
  4. Test and debug using the console
